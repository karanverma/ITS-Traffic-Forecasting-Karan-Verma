# ITS Traffic Forecasting (METR-LA Dataset)

MAC Thesis Interview — METR-LA Traffic Forecasting & Paper Review

## Overview
This project explores traffic flow prediction using the METR-LA dataset. The focus is on understanding temporal traffic patterns and developing baseline predictive models that can support applications in intelligent transportation systems (ITS).

## Dataset
- **METR-LA Dataset**
  - Traffic speed data collected from loop detectors across Los Angeles highways
  - Multivariate time-series data from multiple sensors

## Objectives
- Perform exploratory data analysis (EDA) on traffic data
- Identify temporal patterns and congestion trends
- Build and evaluate baseline predictive models
- Understand key factors influencing traffic behavior

## Methodology
- Data preprocessing and cleaning
- Time-based feature analysis and visualization
- Implementation of baseline and machine learning models
- Evaluation using metrics such as MAE and RMSE

## Key Observations
- Traffic flow shows clear temporal patterns
- Peak-hour congestion is consistently observable
- Model performance depends on feature engineering and data quality

## Limitations
- The current approach mainly focuses on temporal patterns
- Spatial relationships between sensors are not explicitly modeled
- The models used are baseline and can be further improved

## Future Work
- Incorporate spatial dependencies between sensors
- Explore more advanced time-series or deep learning approaches
- Extend the work toward multi-step forecasting
- Study applications in intelligent transportation systems (ITS)
